both 2D and 3D. The calculations for each shape was pre written by the developer.

How to Use - This calculation has a few hot keys forexample -1 can be used to quit the program.
lets take you through how to use:

	calculation.

	After the above you can then select your metric unit of choice then from there
	you will see the calculation screen theis screen is looped unitl the user decides
	to exit and navigate else where.
